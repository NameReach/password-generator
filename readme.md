# Password Generator



A simple interactive Python tool to generate secure passwords within your terminal.



## Features



- Choose password length

- Include/exclude uppercase letters, numbers and special characters

- Generate multiple passwords at once

- See the strength of each Password



The generator can be restarted at any time during the process and can therefore generate as many passwords as required.



## System Requirements



- Python 3.7 or higher

- No extra packages needed



## How to use



Clone the repository and run the script:
```
git clone https://github.com/NameReach/password-generator.git

cd "Password Generator"

python generator.py
```
**Note:** If the "python generator.py" command doesn't work, try right-clicking the app and selecting “Open with” > “Python,” or try launching it using a Python editor like Thonny. That should do the trick.

Then just follow the instructions in the terminal. It should be fairly user-friendly.
